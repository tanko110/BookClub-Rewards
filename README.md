# BookClub-Rewards
This is a GUI based reward points calculator for Serendipity Booksellers. The program computes reward points for customers based on the number of books purchasesd each month. Customers enter the number of books purchased that month, click Computer Points, and then a message box is show that tells the user "Point calculated successfully and instructs the them to click Display Results to see  points earned. The program also has a Clear button, as well as a Quit button to close the program. It includes error handling for null entries and negative numbers.
